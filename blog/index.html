# vuepress博客搭建步骤

  1. 创建一个项目
mkdir 项目名字
  1. 通过npm来初始化项目

注意：在初始化之前要先进入到刚才创建的文件夹

npm init -y
  1. 在LLBlog下面创建一个docs文件夹
mkdir docs
  1. 在docs文件夹下面，创建一个README.md的文件，写一下内容
---
home: true
heroImage: /hero.png
actionText: 快速上手 →
actionLink: /zh/guide/
features:
- title: 简洁至上
  details: 以 Markdown 为中心的项目结构，以最少的配置帮助你专注于写作。
- title: Vue驱动
  details: 享受 Vue + webpack 的开发体验，在 Markdown 中使用 Vue 组件，同时可以使用 Vue 来开发自定义主题。
- title: 高性能
  details: VuePress 为每个页面预渲染生成静态的 HTML，同时在页面被加载的时候，将作为 SPA 运行。
footer: MIT Licensed | Copyright © 2018-present Evan You
---
  1. 在package.json 里面添加两条配置
 "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "dev":"vuepress dev docs",
    "build":"vuepress build docs"
  },
  1. 创建一个.vuepress的文件夹
mkdir .vuepress
  1. 进入到这个.vuepress文件夹,然后创建一个config.js配置
mkdir .vuepresss
cd .vuepress
  1. 在config.js里面增加配置
module.exports={
    title:"大帅比的博客"
}

9.运行 回到项目根目录(LLBlog/),执行下面命令

vuepress dev docs

10.总结

注意：/ -->指向 docs目录 http://localhost:xxx/ 默认去找README.md文件
