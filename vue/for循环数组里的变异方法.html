# for循环数组里的变异方法

变异方法就是指执行后原来的数组被改变 非变异方法就是指执行后原来的数组没有被改变： 变异方法：

a .push() 往数组最后添加一个元素

let arr=[1,333,4,5,6]
this.arr.push(111)

b .pop() 删除数组最后的一个元素

let arr=[1,333,4,5,6]
this.arr.pop()

c .shift() 删除数组的第一个元素

let arr=[1,333,4,5,6]
this.arr.shift()

d .unshift() 在数组的第一个位置添加元素

let arr=[1,333,4,5,6]
this.arr.unshift(1111)

e .splice() 可以删除或者添加元素 第一个参数表示删除或添加元素的开始位置 第二个参数表示删除多少个 第三个参数开始表示添加的元素

let arr=[1,333,4,5,6]
this.arr.splice(1,3,123,6,43)

f .sort() 排序 里面需要传入一个排序规则（函数）默认的是按照字符编码进行排序

升序

let arr=[1,333,4,5,6]
this.arr.sort((a,b)=>{
    return a > b
})

降序

let arr=[1,333,4,5,6]
this.arr.sort((a,b)=>{
    return a < b
})

g .reverse() 反转顺序

let arr=[1,333,4,5,6]
this.arr.reverse()
